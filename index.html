My Coding Journey

Friday, July 11th 2025.

Today marks exactly TWO months since I began my coding journey.

And honestly... if someone had told me I’d be building projects using Java, CSS, and even XML by now, I probably would’ve laughed in disbelief.

Before this, I used to watch coding videos and tutorials — but everything felt like a foreign language. I was just copying code without understanding what it really did. It was frustrating, confusing, and often overwhelming.

But I didn’t give up.

I stayed consistent — through errors, roadblocks, and moments of doubt. Slowly, things started to click. The dots began to connect. I stopped just watching, and I started building.

Where I Started

My very first line of code was written in Python. Although I began with Python, I soon expanded my toolkit—learning to code on my Android phone using Termux and eventually managing my projects with GitHub.

What I’ve Achieved So Far

  • ✅ Built real projects
  • ✅ Collaborated with others
  • ✅ Faced and fixed countless bugs
  • ✅ Actually started enjoying the process of solving them

My Tech Stack

  • Languages: Python, Java, CSS, XML
  • Tools: Termux (on Android), GitHub

My Mission

I hope to inspire other beginners and show that with consistency and curiosity, anyone can become a developer. If you feel like you’re “not getting it”—don’t give up. Sometimes, all it takes is the right guidance and the right space to grow.

Be patient. Stay curious. Keep going. Your breakthrough will come.
